# Movie Recommendation System

This project implements both Content-Based and Collaborative Filtering recommendation systems.

## Features
- Content-based filtering using NLP and cosine similarity
- Collaborative filtering using user similarity
- Built using Python, Pandas, Scikit-learn, and NLTK

## How it works
- Extracts features like genres, cast, keywords
- Converts text into vectors
- Uses cosine similarity to recommend movies

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
